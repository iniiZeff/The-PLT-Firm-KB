Applies To

  • For Bianca and BAELE AND SANDERS SERVICES, LLC
  • All clients with recurring monthly sales tax filing requirements

Filing Deadlines

  • Bianca (Florida): 15th of each month
  • BAELE AND SANDERS (Washington): 20th of each month

Step-by-Step Filing Process

Step 1: Check Assigned Task

  • Go to Monday.com > Recurring Tasks > Monthly Sales Tax
  • Locate the current month‚Äôs task for your assigned client
  • Open their linked Google Sheet
  • Confirm if a sheet exists for this month
  • If not, duplicate the previous month‚Äôs tab and rename it

Step 2: Gather Sales Data

  • Source data from RazorSync, QuickBooks, Client Spreadsheet, or POS system
  • Use filters to set date to previous calendar month
  • Export data to Excel or CSV
  • Remove rows with $0 sales tax
  • Paste into client‚Äôs Google Sheet using columns:
    • Display Name
    • Gross Sales
    • Sales Tax
    • Code (if multi-location)
    • Clean any old/unneeded data

Step 3: Review and Prepare Filing Amounts

  • Sort data by location code
  • Group rows with same code
  • Use SUM formula to total gross per code
  • Prepare list of codes and totals
  • Flat filers: use cleaned monthly gross sales
  • Verify all codes have totals and total matches sum of grouped codes

Step 4: Log In to Filing Portal

  • Retrieve credentials from LastPass
  • Access correct state portal (FL, WA, etc.)
  • Select the return period (previous month)
  • Choose the correct account/entity

Step 5: Input Tax Details

  • Enter Gross Sales and Taxable Amounts
  • Enter code-based totals (for multi-location clients)
  • Apply collection allowance (if applicable)
  • Review for accuracy
  • Proceed to payment screen

Step 6: Submit Payment

  • Use saved banking info
  • ACH Debit ‚Üí Checking ‚Üí Corporate
  • Re-enter payment if prompted
  • Submit the return
  • Download the PDF confirmation

Step 7: Save & Upload

  • Rename PDF using format: Client - [Month Year] Sales Tax Confirmation.pdf
  • Go to: TaxDome > Client > Tax Documents > Sales and Use Tax
  • Upload the confirmation document

Additional Notes

  • Always refer to the client-specific SOP for portal URLs, codes, and payment settings
  • Consult a team lead for address/code guidance
  • Log any unfiled month with reason or delay
